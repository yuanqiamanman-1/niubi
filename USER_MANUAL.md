# 用户手册 (User Manual)

## 1. 快速开始

### 访问系统
打开浏览器访问系统网址（例如：`https://your-site.vercel.app`）。

### 首页（用户预约）
- **填写信息**：输入您的姓名、电话、球拍品牌/型号。
- **选择服务**：填写穿线磅数和羽线型号。
- **提交预约**：点击“模拟支付并提交订单”，系统将生成预约记录。

## 2. 商家管理

### 商家登录
- 访问 `/merchant` 路径（例如 `https://your-site.vercel.app/merchant`）。
- **默认账号**：`admin`
- **默认密码**：`admin123`
- *建议首次登录后修改密码（如系统支持）或在部署时修改数据库初始数据。*

### 订单管理面板
- **待处理订单**：查看所有新提交的预约。
- **状态更新**：点击下拉菜单更改状态（进行中、已完成、取消）。
- **已完成订单**：切换标签页查看历史完成记录，支持按日期/金额筛选导出 CSV。

## 3. 常见问题 (FAQ)

**Q: 提交订单后提示“服务器错误”？**
A: 请检查网络连接。如果是部署版，可能是后端服务正在休眠（Render 免费版会自动休眠），请等待约 1 分钟后重试。

**Q: 如何修改管理员密码？**
A: 目前需通过数据库直接修改，或联系系统运维人员。

**Q: 数据会丢失吗？**
A: 如果后端使用 SQLite 且部署在 Render 免费版，重启后数据可能会重置。建议生产环境配置 PostgreSQL 数据库。
